Description:
This project is a web app that shows real Formula 1 driver and race data in an interactive dashboard. It loads a random upcoming Grand Prix circuit and shows the track image. It also displays current drivers, their teams, headshots, and information in a searchable and sortable table. Finally, a live countdown timer shows exactly how much time is left until the next race.

Why I created it:
I recently got my mom to start watching Formula 1. However, she couldn't keep track of all the drivers. I created this web app to help her memorize the drivers and their teams. It's also mobile friendly, so she can look at it while watching the race on TV.

How it works:
The website uses HTML for structure, CSS for design, and JavaScript for all the logic and interactive stuff. It connects to the OpenF1 API to get real-time data about drivers, teams, and race sessions.
